Projects

  • Uncertainty estimation using deep ensembles
    This project uses deep ensembles, akin to Bayesian neural networks, to better quantify prediction uncertainties in Deep Learning. By training an ensemble of models to predict varying results for the same input, the variance across these predictions measures the uncertainty. High variance, especially for adversarial samples, indicates greater uncertainty.
    • Bayesian neural networks
    • Pytorch
  • Monocular depth estimation
    Estimating depth of a scene with monocular vision using transfer learning. The model uses pretrained densenet as an encoder and a decoder to extract depth from the encoded data. This results in a comparable depth estimation to binocular vision.
    • Python
    • Computer Vision
    • Deep learning
  • Multi agent flocking
    The project deals with a swarm emergent learning system. An approach with multiple autonomous agents which can move in a continuous space by avoiding being caught by a single or multiple predators. The agents use DDPG with awareness of surrounding allies and predators to develop a flocking behavior to avoid the predators.
    • Python
    • Reinforcement learning
    • Pytorch
    • PettingZoo
  • Panorama stitching using gradient matching
    Utilized ORB feature descriptors to stitch multiple images, employing Depth First Search for matching and propagation. Gradient matching was applied to remove differing foreground moving objects in overlapping areas, ensuring preservation of similar gradient image segments.
